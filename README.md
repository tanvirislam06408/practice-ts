# Day One of Learning TypeScript

Today I started learning TypeScript and explored the basics of how it works with JavaScript.

## Summary

- Learned how TypeScript adds static typing to JavaScript.
- Understood the difference between `ts` and `js` files.
- Saw how the TypeScript compiler generates output in the `build` folder.
- Learned about module settings and how browsers use ES modules.
- Practiced a simple TypeScript program with a variable and `console.log()`.
- Learned that `tsconfig.json` controls compilation behavior.

## Day Two of Learning TypeScript

Today I continued learning TypeScript and practiced using more advanced type features.

## Day Two Summary

- Learned about interfaces and type aliases.
- Practiced function typing and optional properties.
- Explored union and intersection types.
- Used generic functions and array typings.
- Understood compile-time error checking in `tsc`.
- Saw how `tsconfig.json` can be adjusted for strict mode and target output.
